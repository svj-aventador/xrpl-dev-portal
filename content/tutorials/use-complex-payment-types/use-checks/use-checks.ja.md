# Checksの使用

SGY LedgerのChecksでは、別のアカウントが後で支払いを請求することが認められていており、個人用の紙の小切手の仕組みと似ています。

**注意:** 2018年10月11日の時点では、[Checks Amendment][]は本番環境のSGY Ledgerで有効になっていません。Checksは[SGY Test Net](xrp-test-net-faucet.html)でのみ使用できます。
